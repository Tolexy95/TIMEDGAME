Welcome to the Number Guessing Game!

*The game will generate a random number between a specified range.

*Your task is to guess the correct number within the given time limit of 60 seconds.

* You can input your guess by entering the number in the provided input field and then clicking the "Guess" button.

* The game will provide feedback on your guess, indicating whether your guess is too high or too low.

* Use this feedback to refine your subsequent guesses and narrow down the possibilities.

* Remember, you must guess the number correctly before the timer runs out to win the game.

* If you guess the correct number within the time limit, congratulations, you win!

* If the timer reaches 0 before you can guess the correct number, the game ends and you lose.

* Now, get ready to put your number-guessing skills to the test and see if you can beat the clock! Good luck!
